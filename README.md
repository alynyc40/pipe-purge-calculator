Pipe Purge Calculator package

Files:
- pipe_purge_calculator.html
- pipe_purge_qr_placeholder.png

How to use:
1. Open the HTML file in any browser to use the calculator.
2. To let welders scan a QR code on their phones, upload the HTML file to your website, SharePoint, Google Drive public link, or any internal web server.
3. Replace the placeholder QR code with a new QR that points to your real hosted URL.

Current QR target:
https://your-domain.example/pipe-purge-calculator.html

Suggested deployment:
- Put the HTML file on an internal company webpage
- Print the QR code and place it near purge stations, welding bays, or tool cabinets

Important:
The QR code I generated is a placeholder because the calculator is not hosted on a public URL yet.
